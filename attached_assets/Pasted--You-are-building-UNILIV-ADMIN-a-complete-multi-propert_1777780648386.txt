  AUTH SYSTEM
  ═══════════════════════════════════════════════════════
  Implement complete JWT auth:
  POST /api/auth/login      → validate credentials, return accessToken (15min) + refreshToken (7d, httpOnly cookie)
  POST /api/auth/refresh    → use refreshToken cookie to issue new accessToken
  POST /api/auth/logout     → invalidate refreshToken in DB
  GET  /api/auth/me         → return current user profile
  
  Middleware:
  - authenticate: verify JWT, attach user to req
  - authorize(...roles): check user role against allowed roles
  - All routes except /api/auth/login require authentication
  
  Password: bcrypt with salt rounds 12
  Seed one SUPER_ADMIN user: admin@uniliv.com / Admin@123
